# 贡献指南

感谢你对 MCP Product Suggester 的关注！欢迎任何形式的贡献。

## 如何参与

1. Fork 本仓库
2. 新建分支（如 `feature/xxx` 或 `fix/xxx`）
3. 提交代码并推送到你的分支
4. 创建 Pull Request，描述你的修改内容

## Issue 反馈

- 欢迎提交 bug、建议或需求，请尽量描述清楚问题场景和复现方式。

## 代码规范

- 遵循 PEP8 编码规范
- 推荐使用 `black` 格式化代码
- 保持注释和文档齐全
- 单元测试需覆盖主要功能

## 交流

如有疑问可通过 Issue 交流，或加入相关社区讨论。
